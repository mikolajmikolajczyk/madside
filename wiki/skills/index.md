canonical forge moved from Radicle to GitHub. The workflow now lives on GitHub
(issues + pull requests); see [`../agents/working-on-issues.md`](../agents/working-on-issues.md)
and the repo-root [`CONTRIBUTING.md`](../../CONTRIBUTING.md).
